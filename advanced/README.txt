Damit der Bot funktionieren kann musst du dir erstmal ein paar pip-Module installieren
1: pip install selenium
2: pip install webdriver_manager
2: pip install PIL (wenn PIL von pip nicht gefunden wird musst du den import in der gui.py rauslöschen.
WICHTIG: Chrome muss installiert sein lol. 


Der Bot kann Mangas nur von einer bestimmten Website herunterladen, achte darauf wenn du einen neuen Manga hinzufügen solltest.

Wenn du sehen willst wie der Bot innerhalb von Chrome navigiert musst du die Chrome_option "--headless" auskommentieren, dann wird Chrome wieder mit UserInterface ausgeführt
